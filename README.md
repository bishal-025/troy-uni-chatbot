# Troy Uni Chatbot

## Introduction
This project is a chatbot designed for Troy University students to assist with inquiries related to university services, events, and information.

## Features
- **Course Information**: Get information about courses offered, schedules, and faculty.
- **Event Updates**: Stay updated with the latest university events and activities.
- **Service Assistance**: Inquire about various student services such as counseling, advising, and registration.
